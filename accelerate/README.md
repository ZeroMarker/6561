# 2048 加速度传感器版

基于手机加速度传感器的 2048 游戏，支持 iOS 和 Android。

## 🎮 在线游玩

访问 [GitHub Pages 链接] 直接开玩！

## 📱 使用方法

1. 点击 **"授权传感器权限"** 按钮
2. 允许权限请求（iOS 会弹出提示）
3. 倾斜手机控制方块移动方向

## 🎯 控制方式

- **📱 倾斜手机** - 前后左右倾斜
- **👆 触摸滑动** - 手指滑动屏幕
- **⌨️ 方向键** - 键盘控制（开启测试模式）

## 🚀 本地开发

```bash
# 启动本地服务器
node server.js

# 访问游戏
http://localhost:3000
```

## 📂 文件结构

```
├── index.html      # 主页面
├── game.js         # 2048 游戏逻辑
├── sensor.js       # 传感器控制（iOS + Android）
├── main.js         # 主程序
├── style.css       # 样式
└── server.js       # Node.js 服务器
```

## 🔧 传感器兼容性

| 平台 | 权限处理 |
|------|---------|
| iOS 13+ | 使用 `DeviceMotionEvent.requestPermission()` 显式请求 |
| Android | 直接添加事件监听器，无需额外权限 |
| 其他 | 自动检测设备方向传感器作为备选 |

## ⚠️ 注意事项

- 需要 HTTPS 环境（GitHub Pages 已支持）
- iOS 用户必须在权限弹窗中点击"允许"
- 如无法使用，请检查浏览器是否支持传感器 API
